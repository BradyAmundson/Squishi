# Squishi!


Features:
- Statically typed
- Type inference
- Simplified String functions using emjis
- Data Structures: Arraylists, linkedlists, simplified superclasses
- List Comprehension

Code Examples
Simplified print statements
```
console.log("Hello, world!");
```
```
🗣️ "Hello, world!"
```
```
